The Main Controller

Controllers are used to organize actions like "show", which can pull data from the database, call code from other places in our app, and renders the desired information on the appropriate web page

                          

  class MainController < ApplicationController


  def show

    @topPromotions = CurrentPromotions.all
    @listing_sample = listings[1..6]
    listings = Listing.all
   
    render :home
  end
end
